# AGENTS.md

## コミュニケーション

日本語の敬語を維持してください。短い依頼でも説明だけで止めず、実装、検証、必要な記録まで進めてください。

## 作業方針

- 作業開始時は branch/worktree を確認し、可能な場合は `git pull --ff-only` を先に実行してください。
- pull-first の同期確認が失敗した場合は更新作業へ進まず、日本語で blocker と再開条件を報告してください。
- 大きい依頼では deep execution を前提に、広く調査し、実装、テスト、ドキュメントまで一気通貫で行ってください。
- 作業完了を最優先し、必要な場合は codex exec や subagent を広く使ってください。
- オートメーション文面や失敗報告は日本語で書いてください。

## ローカルアプリ

この Mac で作業するときは、Codex が開いたウィンドウ、アプリ、ブラウザタブ、プレビュー、Spark Desktop、ChatGPT、Todoist、Discord、Playwright/Chrome などを作業完了前に閉じてください。元から開いていたウィンドウ、未保存の編集、未読メール、未読 DM、未読通知は勝手に閉じないでください。

## メール

メールを使う必要がある場合は、Spark Desktop のみを対象にしてください。Apple Mail、Gmail、Outlook、汎用メールクライアントには切り替えないでください。Codex の作業報告メールの送り先は `yasufumi@koparis.com` のみです。
